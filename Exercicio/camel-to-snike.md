# Camel to Snake

## Dia 2 de dezembro de 2025

Dada uma string em camel case, retorne a versão em snake case da string usando as seguintes regras:

- A string de entrada conterá apenas letras ( A-Ze a-z) e sempre começará com uma letra minúscula.
- Cada letra maiúscula na sequência de caracteres em camel case inicia uma nova palavra.
- Converter todas as letras para minúsculas.
- Separe as palavras com um sublinhado ( __).
