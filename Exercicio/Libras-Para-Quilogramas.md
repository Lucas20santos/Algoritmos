# Libras para quilogramas

## Dia 08 de Dezembro de 2025

## [free Code Camp](https://www.freecodecamp.org/learn/)

Dado um peso em libras como um número, retorne a string "(lbs) pounds equals (kgs) kilograms.".

- Substitua "(lbs)"pelo número inserido.
- Substitua "(kgs)"pelo valor de entrada convertido para quilogramas, arredondado para duas casas decimais e sempre inclua duas casas decimais no valor.
- Uma libra equivale a 0,453592 quilogramas.
- Se a entrada for 1, use "pound"em vez de "pounds".
- Se o valor convertido for 1, use "kilogram"em vez de "kilograms".

## Testes

1. Esperando :1. convert_to_kgs(1)deve retornar "1 pound equals 0.45 kilograms.".
1. Esperando :2. convert_to_kgs(0)deve retornar "0 pounds equals 0.00 kilograms.".
1. Esperando :3. convert_to_kgs(100)deve retornar "100 pounds equals 45.36 kilograms.".
1. Esperando :4. convert_to_kgs(2.5)deve retornar "2.5 pounds equals 1.13 kilograms.".
1. Esperando :5. convert_to_kgs(2.20462)deve retornar "2.20462 pounds equals 1.00 kilogram.".
