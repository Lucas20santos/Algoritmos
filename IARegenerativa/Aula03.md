# Fundamentos do Aprendizado de Máquina

## Tipos de Aprendizados de máquina

### O que é o aprendizado de maquinas

- Treinamentos
    1. Dados de treinamentos: Observações anteriores
    2. Algoritmo: Generaliza a relação entre x e y como uma funcao
    3. Modelo: encapsula a função
- Inteligência
    1. Inferência de dados: recurso naõ rotulados
    2. Predicao: rotulo inferido

### Tipos de aprendizado de máquina

- Aprendizado de máquina:
    - Supervisionado (Dados contém rotulos conhecidos)
        - Regresão (o rotulo é um valor numerico)
        - Classificação (Rotulo é uma categorização ou classe)
            - Classificação Binária: Label é ou não é uma classe
            - Classificação Multiclasse: Label é uma das multiplas classes
    - Não Supervisionado
        - Classificação (rotulo é uma categroização ou classe)
